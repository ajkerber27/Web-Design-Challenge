Summary: Latitude vs. X




The purpose of this project was to analyze how weather changes as location changes in conjunction to the equator. To complete analysis, data from OpenWeatherMap API was pulled to create dataset including more than 500 cities.


Assembled data, used Matplotlip to plot weather v. latitude. Considerations for analysis inlcuded; temperature, cloudiness, humidity, and wind speed. The site provides the source data and visualizations created as part of the data analysis, as well as explanations and descriptions of any trends and correlations witnessed.
